Enter here your code to create shortcuts.

  • Use addAlias("alias_name", "alias_url"); to create a new alias
  • Use addFunction(function); to create a new alias with the function name

    • Note that you can enter any javascript you want.

    Here are some examples.

  • addAlias("exa", "http://www.example.com"); will allow you to go to the given page with "s exa" in the address bar
  • addAlias("w", "https://en.wikipedia.org/wiki/{0}"); will allow you to search in wikipedia with "s w {word}"
  • addFunction(function test(a) { return "http://www.example.com/"+(a*a); }); will redirect you to the given page accordingly to your agruments, e.g. "s test 2"
